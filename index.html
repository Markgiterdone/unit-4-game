Crystal Game


Instructions:


You are provided with a random set of numbers at the start of each game.

There are four crystals below. By clicking on a crystal you will add points to your Current Score.

You win the game by MATCHING your Current Score with the Target Score. And you lose the game if your Current Score exceeds the Target Score.

The value of each crystal is hidden (but you can figure it out by how the score changes).

NOTE: The value of each crystal will randomly change after each game.
